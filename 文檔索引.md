## 🎯 快速動作指南

### 我想...

**🎙️ 立即生成一個播客**
```bash
# 1. 編輯 podcast_config.yaml（設定來源和等級）
# 2. 執行：
python podcast_workflow.py --mode prod
```

**🔧 逐步調試生成過程**  
```bash
python podcast_workflow.py --mode dev
```

**🏷️ 只嘗試標籤嵌入功能**
```bash
# 1. 在 podcast_config.yaml 中設定 tag_embedding.enabled: true
# 2. 執行：
python podcast_workflow.py --mode custom --steps tags --script-dir <existing_script>
```

**⚙️ 修改等級和風格**
- 編輯 `podcast_config.yaml` 中的 `basic.english_level`
- 參考 [configSetting.md](configSetting.md) 了解各等級特色

**🎵 只重新生成音頻**
```bash
python podcast_workflow.py --mode custom --steps audio --script-dir <existing_script>
```
